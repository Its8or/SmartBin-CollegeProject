# _Lixeira inteligente._

### 🧠 Descrição:

O projeto consiste em uma lixeira que abre por aproximação, usando um **sensor ultrassônico**.

### 🔧 Como funciona?

Acoplado à lixeira terá um **sensor ultrassônico**, quando houver movimento a lixeira irá acionar um **motor servo** que abrirá a tampa da lixeira.
