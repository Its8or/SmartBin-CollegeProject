Esse repositório será destinado a um futuro projeto da disciplina "Programação de Microcontroladores".

# _Óculos para auxilio à pessoas com visibilidade nula._

### 🧠 Descrição

O óculos detecta obstáculos no campo de visão e avisam o usuário caso não esteja a uma distância segura.

### ❓ Por que escolhemos esse projeto? 

### 🔧 Como funciona na prática.

O óculos terá acoplado a ele um MCU **ESP32**, responsável pelo funcionamento do projeto. haverão dois sensores **ToF**, que juntos formam um campo de visão de aproximadamente **54°**, para detectar os obstáculos no caminho do usuário. 

Caso o usuário esteja a uma distância não segura do obstáculo, os **drivers hápticos DRV2605L** localizados nas tempuras do usuário emitiram vibrações para avisá-lo.
